🚌 TransitEasy – Public Transport Route Planner

TransitEasy is a smart and user-friendly platform designed to help users find the best way to travel between a source and destination using public transport such as buses, metros, and trains.

---

📌 Overview

TransitEasy simplifies urban travel by providing optimized route suggestions along with useful travel insights. It enables users to quickly identify the most efficient way to reach their destination while also exploring nearby attractions.

The platform focuses on making travel simple, efficient, and accessible for everyone.

---

✨ Key Features

✔ Smart route planning with best and alternative routes
✔ Estimated travel time and cost for each route
✔ Multiple transport modes (Bus, Metro, Train, Mixed)
✔ Recommended route highlighting for easy decision-making
✔ Interactive route path visualization
✔ Built-in tourist place suggestions
✔ Accessibility-friendly travel information
✔ Simple and clean user interface

---

🚏 Route Options

- 🚇 Metro Route (Fastest)
- 🚌 Bus Route (Cheapest)
- 🚇 + 🚶 Recommended Route
- 🚌 + 🚖 Mixed Route (if no direct transport)

---

♿ Accessibility Support

TransitEasy ensures inclusive travel by providing details such as:

- Wheelchair accessible routes
- Elevators and ramps
- Low-floor buses
- Easy navigation for differently-abled users
